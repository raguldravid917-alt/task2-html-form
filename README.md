# Task 2 - HTML Form

This project is created as part of my HTML practice task.
It contains a basic user form built using only HTML.

Features
- Name input
- Email input
- Gender selection (Radio buttons)
- Skills selection (Checkboxes)
- Country dropdown
- Message textarea
- Submit and Reset buttons

Technologies Used
- HTML only (No CSS, No JavaScript)
